### Run app without docker

```
git clone https://github.com/fadilrisdian/streamlit-car-counting
cd streamlit-car-counting
pip install -r requirements.txt
streamlit run App.py
```

### Build application
```
git clone https://github.com/fadilrisdian/streamlit-car-counting
docker build ./
docker run (image id)
```
